<h1 align="center">Тестовое задание "Beauty Store - Мобильное приложение для магазина косметики"</h1>

<h2 align="center">Описание"</h2>
Beauty Store - мобильное приложение для онлайн-магазина косметики и уходовой продукции. Приложение предоставляет персонализированный подход к подбору косметики.

<h2 align="center">Основные возможности</h2>

<h3 align="center">Визуальные особенности</h3>
•	Вёрстка по макетам с Figma

<h2 align="center">Технологии</h2>
•  Flutter

•  Dart

•  Material Design
